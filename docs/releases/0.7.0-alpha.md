# REINDUSTRIALIZE 0.7.0-alpha

Release channel: first playable alpha

## Playable now

- Create and name a founder and manufacturing company.
- Learn the Garage Bay from Zach through narrated story and station guidance.
- Accept customer work, order and cut stock, set tools, run true G/M-code exercises, inspect parts, hire a team, and reach the Job Shop arrival.
- Use keyboard, Xbox-compatible controller, touch controls, or the phone QR controller.
- Save locally without creating an account.

## Release improvements

- Real human-bot gameplay footage and narrated landing-page video.
- Search discovery package: crawlable robots file, sitemap, canonical metadata, favicon/app icons, web manifest, and `llms.txt`.
- Secure, disabled-by-default Supabase preparation with local saves remaining primary.
- Versioned `/release.json` for comparing the deployed build with source.
- Amazon Luna/Appstore pre-authorization submission bundle and certification checklist.
- Dedicated mobile viewport gameplay audit and regression coverage.

## Honest scope

Chapter 1 through the Job Shop arrival is the current playable campaign slice. Chapters 2–6 remain development/planned content and are not represented as shipped gameplay.
