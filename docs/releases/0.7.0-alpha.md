# REINDUSTRIALIZE 0.7.0-alpha

Release channel: first playable alpha

## Playable now

- Create and name a founder and manufacturing company.
- Learn the Garage Bay from Zach through narrated story and station guidance.
- Accept customer work, order and cut stock, set tools, run true G/M-code exercises, inspect parts, hire a team, and reach the Job Shop arrival.
- Use keyboard, Xbox-compatible controller, touch controls, or the phone QR controller.
- Save locally without creating an account.
- Select missions directly; locked steps explain and route through their next safe prerequisite.
