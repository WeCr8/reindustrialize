# REINDUSTRIALIZE Remotion Demo

The 60-second `ReindustrializeDemo` composition uses deterministic Playwright screenshots and the generated ElevenLabs `founding_dawn` game cue.

```powershell
pnpm demo:studio
pnpm demo:render
```

Regenerate screenshots with `python scripts/capture-demo-screenshots.py` while the local game server runs on port 8787.
