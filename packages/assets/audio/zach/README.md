# Zach Voice Library

Approved, reusable Zach Goodbody voice assets for gameplay. Captions remain authoritative and every clip must be selected by its stable ID rather than by filename guessing.

- Story/task-specific clips and generation text: `data/audio-generation.json`
- Reusable response IDs, tags, and intended contexts: `data/zach-reusable-voice-clips.json`
- Consent and use boundaries: `docs/ZACH_VOICE_CONSENT.md`

Do not synthesize arbitrary player-provided text, use these files outside the approved project scope, or replace Zach's mentorship with unreviewed dynamic dialogue.
